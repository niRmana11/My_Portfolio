BookUs â€“ Appointment Booking & Management System

A full-stack appointment booking & management system.

BookUs is a full-stack web platform designed to simplify appointment scheduling between service providers and customers. It offers an intuitive interface where providers manage services and time slots, while customers can book, edit, and track appointments seamlessly.

Key Features:

Customer Side
  • Browse services by category
  • Book, edit, or cancel pending appointments
  • View appointment history
  • Profile management (name, email, password)
Provider Side
  • Create & manage services with price, duration, and category
  • Auto-generate time slots based on service duration
  • Manage appointments with status updates (Pending, Confirmed, Completed, Cancelled)
  • Profile management for providers

Tech Stack:

  • Backend: Java 22, Spring Boot, Spring Data JPA, Hibernate, MySQL
  • Frontend: React.js, Bootstrap 5, Axios, React Router
  • Tools: VS Code, Postman, GitHub, MySQL Workbench
